Typing Speed Tester

A simple and interactive Typing Speed Tester built using HTML, CSS, and JavaScript. This project helps users measure their typing speed (WPM - Words Per Minute) and accuracy in real time.

📌 Features
⌨️ Real-time typing test
⏱️ Timer-based typing challenge
📊 Words Per Minute (WPM) calculation
🎯 Accuracy tracking
🔄 Restart test functionality
🎨 Clean and responsive UI
🛠️ Tech Stack
HTML – Structure of the application
CSS – Styling and responsive design
JavaScript – Logic, timer, and calculations

⚙️ How It Works
User starts typing the given text.
Timer begins automatically.
System tracks:
Time taken
Correct characters
Incorrect characters
At the end, it calculates:
WPM (Words Per Minute)
Accuracy (%)

🧮 Formula Used
WPM (Words Per Minute)
WPM = (Total Characters / 5) / Time (in minutes)
Accuracy
Accuracy = (Correct Characters / Total Typed Characters) × 100
