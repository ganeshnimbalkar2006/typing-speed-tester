
# 🚀 Typing Speed Tester  

A simple web application that helps users check their typing speed and accuracy in real time. It provides instant feedback while typing, making it useful for improving typing skills.
---
## 📌 About the Project  
This project displays a paragraph for users to type and measures how fast and accurately they complete it. It shows results like words per minute (WPM), accuracy, and mistakes instantly.
---
## ✨ Features  
- ⏱️ Real-time typing speed (WPM)  
- 🎯 Accuracy tracking  
- ❌ Mistake highlighting  
- 🔄 Restart test option  
- 📱 Responsive design  
---
## 🛠️ Technologies Used  
- HTML  
- CSS  
- JavaScript  
---
## 🎯 Purpose  
- Improve typing speed  
- Practice accuracy  
- Prepare for typing tests
